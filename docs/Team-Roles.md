# Team Roles

- Product Lead: Owns roadmap, prioritization, and stakeholder comms.
- Tech Lead: Owns architecture, code quality, and releases.
- Backend Engineers: Implement controller/nodes features.
- Frontend Engineers: Build web-ui UX and integrations.
- QA/Testing: Define and execute tests, performance and reliability.
- DevOps: CI/CD, deployments, observability, cost management.
